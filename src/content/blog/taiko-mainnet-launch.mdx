We are thrilled to announce that Taiko Mainnet is officially live on Ethereum. This marks a significant milestone for the Taiko project and the broader Ethereum ecosystem.

## What This Means

Taiko is now a fully operational based rollup on Ethereum, inheriting its security, liveness, and credible neutrality. With our Type 1 ZK-EVM, developers can deploy existing Ethereum smart contracts without any modifications.

## Key Features at Launch

- **Based Sequencing**: Transaction ordering is handled by Ethereum validators
- **Permissionless Proposing**: Anyone can propose blocks
- **Permissionless Proving**: Anyone can prove blocks
- **Full EVM Equivalence**: All Ethereum tooling works out of the box

## Getting Started

To start using Taiko Mainnet:

1. Bridge your ETH using the [Taiko Bridge](https://bridge.taiko.xyz/)
2. Configure your wallet with the Taiko RPC
3. Deploy your contracts using your favorite tools

We look forward to seeing what the community builds on Taiko!
