## What was done

1. **Hero** — Full viewport section with animated title ("a based rollup"), two CTA buttons (Start building, Bridge to Taiko), TaikoIcon 3D logo with GSAP scroll parallax to scaling section
2. **Scaling** — Horizontal scroll with 4 pinned 100vw screens using GSAP ScrollTrigger, SequentialDots canvas animation, Rays SVG stroke animation, taikoIconAnimation for Taiko/Ethereum icon alternation
3. **Explore** — Split-screen Alethia/Gwyneth with hover state toggling, ScrollTrigger pin, rotating Taiko logo using static SVG assets
4. **About** — AboutGrid SVG stroke reveal on scroll, TextSplitter word-by-word opacity animation via GSAP scrub
5. **Journey** — JourneyPath SVG with stroke-dashoffset reveal, 6 absolutely-positioned JourneyItem components (Q2 2024 → H1 2025) with skewY transforms and expandable content
6. **Governance** — Dot pattern background, governance video with mix-blend-mode darken, title/description
7. **Ecosystem** — Reuses existing Banner (Trailblazer) and GrantsBanner (Grant Factory) components
8. **Subscribe** — Wraps existing FooterSubscribe newsletter component
