### S08: Blog Listing & Article Pages
Status: in_progress
Depends on: S05

Description:
Implement the Blog listing page (/blog) with paginated grid of blog cards (image, title, date, timeToRead, category), category filter sidebar, date filter dropdown, and pagination. Implement the Blog article page (/blog/[slug]) with article header (title, date, timeToRead, category, image), MDX article body rendering, optional How to Apply section, latest articles sidebar, and previous/next navigation. Use blog data from src/content/blog/ MDX files and src/content/data/blog-categories.ts. Content utility functions should handle pagination and filtering.

Acceptance Criteria:
- [ ] Blog listing shows paginated grid of blog posts
- [ ] Category filter works to narrow displayed posts
- [ ] Date filter dropdown works
- [ ] Pagination navigates between pages
- [ ] Blog article pages render MDX content correctly
- [ ] Previous/next navigation works between articles
- [ ] Responsive layout works across all breakpoints
- [ ] pnpm run build succeeds
